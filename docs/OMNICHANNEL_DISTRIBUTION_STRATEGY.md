## Part 4: BlueSky Strategy

- 62% under 34
- Tech-savvy, privacy-focused
- Early adopters, open-source nerds

### Content Approach

| What Works | What Doesn't |

### Optimal Posting
- **Days:** Wednesday, Thursday
- **Times:** 9am-noon, 3-5pm (target timezone)
- **Format:** Short insights, questions, personality

### Community Building
- Engage with Techsky hashtags
- Participate in developer discussions
- Share open-source contributions

| **r/coolgithubprojects** | 60K | OSS friendly | GitHub releases |

### Golden Rules

1. **10:1 Rule:** 10 valuable comments for every self-promotional post
2. **Value First:** Educational content, not sales
3. **Community Member:** Be a regular, not a drive-by
4. **Format Matters:** Use Reddit formatting, not links only

## Part 9: Static Website CMS Strategy

### Recommended Stack

| Component | Choice | Why |
|-----------|--------|-----|
| **SSG** | **Astro** | Content-focused, islands architecture |
| **Alt SSG** | Hugo | Blazing fast builds |
| **Headless CMS** | Contentrain or Sitepins | Git-based, free |
| **Hosting** | VPS + Nginx | Full control |
| **CDN** | Cloudflare | Free tier, global |

### Astro Advantages

- Partial hydration (minimal JS)
- Multi-framework support
- Content Collections API
- Starlight for docs

### Content Types Supported

| Type | Format | Location |
|------|--------|----------|
| Blog posts | Markdown | /blog |
| Docs | MDX | /docs |
| Changelog | Markdown | /changelog |
| Tutorials | MDX + video embeds | /tutorials |
