# OSS vs. Pro Module Allocation Strategy

**Version:** 1.0 (Pre-Launch Finalization)
**Date:** 2025-12-29
**Goal:** Maximize ubiquity (OSS) while protecting sustainability (Pro).

---

## The Philosophy: "Primitives vs. Solutions"

- **Open Source (Primitives):** The fundamental building blocks of reasoning. If it's a "cognitive action" (e.g., "deduce," "critique," "verify"), it belongs in the core. We want these to become the standard library of AI.
- **Pro (Solutions):** The orchestration of primitives into complex, business-specific workflows. If it's a "business outcome" (e.g., "Risk Analysis," "Decision Matrix," "Scientific Research"), it belongs in Pro.

---

## 1. The OSS Core (Apache 2.0)

_Free forever. The hook._

| Module            | Function            | Why OSS?                                           |
| :---------------- | :------------------ | :------------------------------------------------- |
| **GigaThink**     | Divergent Thinking  | Basic brainstorming. Table stakes.                 |
| **LaserLogic**    | Deductive Reasoning | Logic is infrastructure. Must be free.             |
| **BedRock**       | First Principles    | Breaking things down is fundamental.               |
| **ProofGuard**    | Verification        | Reliability is our brand. Can't gate safety.       |
| **BrutalHonesty** | Critique            | Self-correction is essential for any usable agent. |

**Strategy:** These 5 modules allow a developer to build _anything_. They are the "LEGO bricks." By making them free, we prevent competitors from undercutting us on basic capability.

## 4. The "Trojan Horse" Strategy

1.  **Individual developers** install OSS ReasonKit to fix their own flaky prompts.
2.  **They get addicted** to the "ProofGuard" reliability and the SQLite audit trails.
3.  **Their boss asks** "Why did the AI say that?"
4.  **Developer shows** the local audit log.
5.  **Boss asks** "Can we have this for the whole team?"
6.  **Developer says** "We need the Pro version for centralized logging and SSO."
7.  **Sale.**
