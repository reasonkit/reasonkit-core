Standard LLMs hallucinate. ReasonKit **ThinkTools** enforce specific cognitive operations.

| Module | Function | Description |
|--------|----------|-------------|
| **GigaThink** | `Diverge()` | Forces 10+ distinct perspectives to prevent tunnel vision. |
